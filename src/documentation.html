API for start & end location

API for array of locations

https://best-way.herokuapp.com/api/locations/:locations

PARAMS: locations (JSON stringified array) → Array[STRING]

API for wetaher based on one location by name

https://best-way.herokuapp.com/api/weather/:cityname

EXAMPLE: https://best-way.herokuapp.com/api/weather/ljubljana
REQUEST TYPE: GET
PARAMS: location name → STRING
RESPONSE: Weather object for single location
Reponse example:
   { 
        {
            "ok":true,
            "result":{
                "location":{
                    "longitude":14.51,
                    "latitude":46.05
                },
                "weather":{
                    "temp":279.44,
                    "wind":2.1,
                    "country":"Slovenia",
                    "city":"Ljubljana",
                    "main":"Clouds",
                    "description":"overcast clouds",
                    "pressure":1020,
                    "humidity":87
                }
            }
    }

API for wetaher based on one location by geolocation

https://best-way.herokuapp.com/api/weather/:lat/:lon

EXAMPLE: https://best-way.herokuapp.com/api/weather/46.23092/15.26044
REQUEST TYPE: GET
PARAMS: latitude → DOUBLE, latitude → DOUBLE
RESPONSE: Weather object for single location than 5 items in array
Reponse example:
   {
        "ok":true,
        "result":{
            "location":{
                "longitude":15.26,
                "latitude":46.23
            },
