Provided routes by Best-Way REST API

List of APIs

API for start & end location
API for array of locations
API for wetaher based on one location by name
API for wetaher based on one location by geolocation

API for start & end location

https://best-way.herokuapp.com/api/between/:start/:end

EXAMPLE: https://best-way.herokuapp.com/api/between/ljubljana/vienna
REQUEST TYPE: GET
PARAMS: start → STRING, end → STRING
RESPONSE: 5 inbetween locations with the highest temp

API for array of locations

                
https://best-way.herokuapp.com/api/locations/:locations

                

                

                    EXAMPLE: best-way.herokuapp.com/api/locations/["celje","praga","ljubljana",
                        "koper"]

                    REQUEST TYPE: GET

                    PARAMS: locations (JSON stringified array) → ARRAY[STRING] 

                    RESPONSE: Weather for all locations or 5 inbetween locations with the highest temp if more
                    than 5
                    items in array

API for wetaher based on one location by name

                
https://best-way.herokuapp.com/api/weather/:cityname

                

                

                    EXAMPLE: https://best-way.herokuapp.com/api/weather/ljubljana

                    REQUEST TYPE: GET

                    PARAMS: cityname → STRING 

                    RESPONSE: Weather object for single location (search is based on location name)

https://best-way.herokuapp.com/api/weather/:lat/:lon

                    REQUEST TYPE: GET

                    PARAMS: lat → DOUBLE, lon → DOUBLE
